## Keybindings

Keybindings | Action
--------|-------
c | Clear
Backspace | Undo
s | Insert sign
p | Insert parenthesis
\+ | Add
\- | Subtract
\* | Multiply
/ | Divide

## Expression

Operator | Operation
---------|----------
\* | Multiplication
/ | Division
\+ | Addition
\- | Subtraction

## Warnings

This is a single-threaded application which does not provide any failsafe for expressions that might are too big to compute
